SUBJECT: A complete 16×16 pixel-art tile atlas for the "<BIOME_DISPLAY_NAME>" biome — <ONE_SENTENCE_BIOME_DESCRIPTION>. The atlas contains floor tiles, an autotile transition set, capped wall tiles, ore tiles, and decoration tiles arranged on a 16×16 grid (256 total cells).

STYLE: 16-bit pixel art, top-down 2D with a slight 3/4 lean. Hard pixel edges, nearest-neighbor scaling, no anti-aliasing, no gradient ramps, no soft shadows, no per-tile perspective. Inspired visually by Core Keeper, Stardew Valley, and Songs of Conquest pixel-art density.

COMPOSITION: A 256×256 pixel image divided into a 16×16 grid; each cell is exactly 16×16 pixels with no gaps, padding, or borders between cells. Each cell is a self-contained tile whose content fills its 16×16 bounding box.

SIZE: Generate at 1024×1024 source resolution. Final asset is 256×256 (4× nearest-neighbor downsample). Compose so the 16×16 grid is exactly preserved after downsample — each tile is 64×64 in the source image.
