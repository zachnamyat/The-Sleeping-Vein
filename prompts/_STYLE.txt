# SHARED STYLE PREAMBLE
#
# Every Gemini prompt in this directory implicitly inherits the STYLE and ANTI-PROMPT
# blocks below. When you assemble a prompt for the model, prepend these blocks to the
# per-asset file's content unless that file overrides them.

STYLE: 16-bit pixel art, top-down 2D with a slight 3/4 lean. Hard pixel edges, nearest-neighbor scaling, no anti-aliasing, no gradient ramps, no soft shadows. Inspired visually by Core Keeper, Stardew Valley, and Songs of Conquest pixel-art density. Pixel art should remain crisp and readable when downsampled to its target size.

ANTI-PROMPT: photorealism, soft shadows, blur, anti-aliasing, gradient skies, sub-pixel detail, watermark, signature, text labels (unless the asset IS a text/script piece), frame, border, isometric perspective, JPG compression artifacts, color bleed across cell boundaries, mismatched cell sizes in sheets, multiple disjoint subjects per cell.
